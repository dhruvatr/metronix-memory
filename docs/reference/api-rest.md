# REST API Reference

The detailed REST API reference lives in [`../API.md`](../API.md).

Main endpoint groups:

- `/api/v1/documents`
- `/api/v1/connections`
- `/api/v1/files`
- `/api/v1/memory`
- `/api/v1/agents`
- `/api/v1/knowledge`
- `/api/v1/traces`
- `/api/v1/workspaces`

Use REST when you need direct application integration rather than an MCP client.
